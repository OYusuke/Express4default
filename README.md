#express4 template for ect

## How to Use

### install and edit
```
$ git@github.com:OYusuke/Express4default.git
$ cd Express4default
$ npm i
$ DEBUG=Express4default ./bin/www				//Mac
$ set DEBUG=Express4default & node .\bin\www	//Win
  myapp Express server listening on port 8080 +0ms
```

### access

http://localhost:8080
